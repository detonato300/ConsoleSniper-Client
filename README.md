# Console Sniper v3.1.0 🚀

**Console Sniper** is a high-performance, distributed market intelligence tool designed for 24/7 monitoring of Japanese marketplaces (Mercari, Yahoo Auctions/Buyee). It specializes in identifying high-value "Junk" consoles and mid-range deals using AI-driven risk assessment.

## Key Features

- **Distributed Orchestration:** Leveraging a network of volunteer nodes for high-speed, stealthy scraping.
- **AI Analysis:** Integrated Gemini 1.5 Pro/Flash logic for deep-dive technical condition assessment.
- **End-to-End Encryption (E2EE):** All communications are secured with AES-256-GCM and HMAC signatures.
- **Modern TUI:** A fast, interactive terminal interface built with Bubble Tea and Lipgloss.
- **Bounty System:** Earn points and level up by contributing to the network's data ingestion.

## Requirements

- **Operating System:** Windows, Linux, or macOS.
- **Network:** Stable internet connection (Residential IP preferred for volunteer mode).
- **License:** A valid Console Sniper license key (Redeemable in the app).

## Quick Start

1. Download the binary for your system:
   - **Windows:** `consolesniper.exe`
   - **Linux:** `consolesniper_linux_amd64`
   - **macOS (Intel):** `consolesniper_darwin_amd64`
   - **macOS (Apple Silicon M1/M2/M3):** `consolesniper_darwin_arm64`
2. **Linux/macOS:** Grant execution permissions:
   ```bash
   chmod +x consolesniper_target_file
   ```
3. Launch the application.
3. Go to the **License** tab and paste your activation key.
4. Enable **Volunteer Mode** in the Dashboard to start earning points.
