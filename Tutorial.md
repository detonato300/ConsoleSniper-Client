# Console Sniper - User Tutorial 📖

Welcome to the **Console Sniper** ecosystem. This guide will help you set up and maximize the potential of your sniping operations.

## 1. Installation & First Run

- **Executable:** Simply run `consolesniper.exe`.
- **First Run:** On the first run, the app will generate a unique **HWID** (Hardware ID) bound to your machine. 
- **License Activation:**
    1. Navigate to the **License** tab using the arrow keys or `[1]`.
    2. Press `[Enter]` to open the activation prompt.
    3. Paste your key (Ctrl+V) and press `[Enter]`.
    4. Once activated, your license is bound to your HWID.

## 2. Interface Navigation

The interface is divided into several tabs:
- **[1] License:** Manage your activation and subscription tier.
- **[2] Dashboard:** Your operational center. Monitor your Level, XP, and active tasks.
- **[3] Search:** Perform private searches (Premium feature).
- **[4] Settings:** Configure AI keys and view system info.

## 3. Volunteer Mode (The Heart of the System)

By enabling **Volunteer Mode** (Press `[V]` on the Dashboard), you contribute your local IP to the distributed network.
- **Tasks:** You will automatically receive and process discovery/search tasks.
- **Rewards:** Every successfully processed task grants **XP** and **Points**.
- **Bounties:** High-priority tasks (Bounties) provide massive point multipliers.
- **Stealth:** The system uses randomized polling intervals (45-60s) to ensure safe operation.

## 4. Advanced Features

### AI Deep-Dive
If you have a Gemini API key configured in **Settings**, you can unlock advanced analysis features in the Discord Command Center.

### Local Search
Premium users can perform "Stealth Searches". These queries are processed locally and results are not shared with the global database, ensuring your "private goldmines" stay private.

### Debug Mode
For advanced troubleshooting, you can launch the client with a secret flag to unlock the hidden **Debug** tab:
```powershell
.\consolesniper.exe --DEBUGDEVSNIPER2026
```

## 5. Troubleshooting

- **Poll Error:** Usually means the server is undergoing maintenance or your internet connection is unstable.
- **HWID Conflict:** Licenses are locked to one machine. Contact an administrator (<@756558181756436500>) to reset your HWID.
